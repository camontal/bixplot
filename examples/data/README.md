# Data sources

This folder contains datasets used for demonstration and examples in the `bixplot` package.

These datasets originate from a mix of published research and publicly available datasets.
They are included for illustration purposes only.

---

## AVONET dataset

- Source: Tobias, J. A. et al. AVONET: morphological, ecological and geographical data for all birds. Ecology Letters 25, 581–597 (2022).
- Description: Global dataset of bird traits (e.g., beak size, body mass, wing length)

---

## BehaviourTrials dataset

- Source: Kerr, N. R. & Ingram, T. Personality does not predict individual niche variation in a freshwater fish. Behav Ecol 32, 159–167 (2021).
- Description: Behavioral fish data
