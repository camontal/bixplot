# bixplot

The bixplot: A variation on the boxplot suited for bimodal data





\*\*The bixplot\*\* is a variation of the classical boxplot designed to detect and display bimodality and multimodality in univariate data, while combining the strengths of the boxplot, violin plot, and beanplot by integrating a central box representing the interquartile range, density curves, and individual data visualization.



---



\## Folder structure



\### `bixplot/`

Contains the core Python module implementing the bixplot display and related functions.

\- `bixplot` – main plotting functions

\- `colors` – function related to the color schemes and styling

\- `pamc1d` – constrained clustering algorithm used to detect contiguous clusters



\### `examples/`

Includes example scripts and notebooks illustrating bixplot applications.

\- `paper visuals python.ipynb` – examples of figures on various datasets
